A small, reliable Python YouTube downloader offering a GUI and a CLI.
Python | CLI | GUI

Overview

PYTdwnloader is a lightweight tool I developed to fetch YouTube videos for editing and offline use. It has both a graphical interface and a command-line option.

How to use

If you are on windows simply run the .exe file and you will be on your way.

If you are on linux simply go to the github download the source and run the build script an you will be on your way.
